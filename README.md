# Scikit-learn 100+ Exercises 🧠

This repository contains my solutions and practice work from the Udemy course **"Scikit-learn in Python: 100+ Data Science Exercises"**.

The goal is simple: build a solid, hands-on understanding of machine learning by actually solving problems — not just watching theory.

---

## 📚 What’s inside

The exercises cover a wide range of machine learning topics, including:

- Data preprocessing (missing values, encoding, scaling)
- Feature engineering and selection
- Classification & regression models
- Clustering algorithms
- Model evaluation metrics
- Hyperparameter tuning (GridSearchCV)
- Dimensionality reduction (PCA)
- Text processing (CountVectorizer, TF-IDF)
- Outlier detection
- Ensemble methods
